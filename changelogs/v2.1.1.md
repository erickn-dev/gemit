## CHANGELOG

Este CHANGELOG documenta as mudanças recentes no projeto, incluindo a configuração inicial com TypeScript e LLM, diversas melhorias na interface de linha de comando (CLI), mecanismos de atualização e aprimoramentos na interação com IA.

### Features

*   Configuração inicial do projeto com TypeScript, módulo LLM e script de build seguro.
*   Introdução da integração central LLM e estrutura do projeto.
*   Adição de novos comandos CLI e aprimoramento do comando `commit`.
*   Adição do comando `add` e novas opções para o comando `commit`.
*   Implementação de limite de commits para o changelog e renomeação do pacote.
*   Adição de mecanismo de atualização automática.
*   Modernização da interface de usuário (UI) e melhoria da atualização automática no Windows.
*   Adição de comando de atualização manual.
*   Otimização do fluxo de confirmação de mensagens de commit geradas por IA.
*   Adição do comando `update` e melhoria do fluxo da UI.
*   Adição de prompts de IA personalizáveis e redesenho da interface CLI.
*   Adição de configurações locais e permissões para o módulo Claude.
*   Adição de novos comandos e melhoria de funcionalidades centrais.

### Fixes

*   Atualização da versão do pacote para 1.8.2.

### Refactors

### Docs

*   Remoção de arquivos de changelog desatualizados.

### Chore
