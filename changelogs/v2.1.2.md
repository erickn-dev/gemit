# CHANGELOG

Este CHANGELOG documenta as mudanças recentes no projeto, incluindo novas funcionalidades, correções de bugs, refatorações e atualizações de manutenção.

## Features

-   **Configuração Inicial e Integração LLM:**
    -   Configuração inicial do projeto com TypeScript, módulo LLM e script de build seguro.
    -   Introdução da integração central de LLM e estrutura do projeto.
-   **Melhorias na CLI e Comandos:**
    -   Adicionados novos comandos CLI e aprimoramento do comando `commit`.
    -   Adicionado o comando `add` e novas opções para `commit`.
    -   Adicionado limite de commits para o changelog e renomeado o pacote.
    -   Adicionado comando de atualização manual.
    -   Adicionado comando `update` e melhorado o fluxo da interface do usuário.
    -   Adicionados novos comandos e melhorias nas funcionalidades principais.
-   **Atualização Automática e UI:**
    -   Adicionado mecanismo de atualização automática.
    -   Modernizada a interface do usuário e melhorada a atualização automática no Windows.
-   **Inteligência Artificial:**
    -   Simplificada a confirmação de mensagens de commit geradas por IA.
    -   Adicionados prompts de IA personalizáveis e redesenhada a interface CLI.
    -   Adicionadas configurações locais e permissões para Claude.
