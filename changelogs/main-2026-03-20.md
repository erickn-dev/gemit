# Changelog - 1.7.0 (2026-03-20)

## 🆕 New Features & Improvements

### 🎨 Modernized UI
- **Refined Visual Style:** Complete UI redesign with a more modern and elegant aesthetic.
- **Unicode Symbols:** Replaced text labels with clear symbols (`✔`, `⚠`, `✖`, `ℹ`, `›`).
- **Enhanced Color Palette:** Integrated `gray`, `magenta`, `italic`, and `underline` for better hierarchy.
- **New Spinner:** Smooth dot-based animation for background tasks.
- **Smart Headers:** Prominent section headers for better readability.

### 🔄 Improved Auto-Update (Windows)
- **Detached Windows Update:** On Windows, the update now opens in a new, separate terminal window. 
- **Increased Reliability:** This prevents issues related to file locks when updating the package while it's in use.
- **Visual Feedback:** The new update window stays open after completion so you can see the results.

## 🛠 Fixes
- **npm execution:** Fixed internal npm execution on Windows to handle specific environment constraints.
- **Label Formatting:** Standardized status labels across all CLI commands.
