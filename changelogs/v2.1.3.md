# CHANGELOG

Este changelog resume as mudanças recentes no projeto, incluindo novas funcionalidades, correções, refatorações e atualizações de manutenção.

## Features

-   Configuração inicial do projeto com TypeScript, módulo LLM e script de build seguro.
-   Introdução da integração central LLM e estrutura do projeto.
-   Adição de novos comandos CLI e melhorias no comando `commit`.
-   Adição do comando `add` e novas opções para `commit` na CLI.
-   Implementação de limite de commits para changelog e renomeação de pacote na CLI.
-   Adição de mecanismo de atualização automática.
-   Modernização da interface do usuário e melhoria da atualização automática no Windows.
-   Adição de comando de atualização manual.
-   Otimização da confirmação de mensagens de commit geradas por IA.
-   Adição do comando `update` e melhoria do fluxo da UI.
-   Adição de prompts de IA personalizáveis e redesenho da interface CLI.
-   Adição de configurações locais e permissões para Claude.
-   Adição de novos comandos e melhoria das funcionalidades principais.
